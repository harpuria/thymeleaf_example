# thymeleaf_example
현업에서 JSP 만 주구장창 쓰다가 이참에 타임리프를 공부하기 위해 만든 저장소입니다. 

***

[목표]
+ 타임리프 예제 정리

***

[개발 환경]
+ JAVA (11, Zulu Open JDK)
+ Spring Boot (2.3.4)
+ Thymeleaf (2.3.4)
+ Gradle (6.5.1)
+ IntelliJ Community (2020.2.1)
+ BootStrap (4.5.2)

***

[참고 자료]
+ 타임리프 공식 홈페이지 [(타임리프 바로가기)](https://www.thymeleaf.org/ "타임리프 바로가기")
+ 구글
